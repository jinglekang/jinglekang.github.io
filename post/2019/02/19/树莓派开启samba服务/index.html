树莓派开启samba服务

段落导航

声明：debian 9.0系统，如未登录root，请自行添加sudo

首先更新系统

apt update
apt upgrade

安装samba程序

apt install samba samba-common-bin

编辑配置文件

nano /etc/samba/smb.conf

在末尾加入下面代码：

[public] # 共享名称
  comment = Public Storage # 备注
  path = /public # 共享路径
  public = yes # 是否公开（匿名访问，Ghost）
  browseable = yes # 是否可以浏览
  writable = no # 是否可以写入

上面的是添加一个public共享，允许匿名访问，不可修改。

如果需要添加一个需要登录并运行修改的，可以这么写：

[private]
  comment = Private Storage
  path = /public
  public = no
  browseable = yes
  writable = yes
  valid users = pi # 允许pi用户访问共享

上面的是指允许pi用户进行登录和修改，然后给pi用户设置密码，注意，pi用户可以是任意linux系统用户。

smbpasswd -a pi

输入两次密码后，重启服务，就可以正常使用了。

/etc/init.d/samba restart
