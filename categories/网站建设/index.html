解决WordPressg更新或安装插件需要FTP账号密码的问题

无法安装插件或无法更新是因为apache没有权限，直接把WordPress安装目录所属人转移apache即可。

域名无故被serverHold

前段时间收到腾讯企业邮箱的通知，内容大约是我的备用域名（div.ltd）MX记录没了。MX记录有大半年没改过了，不可能无缘无故消失，吓我还以为域名被盗，赶紧去NameSilo查看，但是没有发现问题，就没有放在心上，连续几天通知我记录丢失，才注意到域名状态是serverHold了。

div.ltd域名已经转移到NameSilo了

放了很久的域名，也没有挂网站，但是不舍得扔，只是解析到Google Cloud服务器，前几天准备测试用，结果发现怎么调整都打不开网站，以为是服务器apache问题，就准备用阿里云的服务器测，不过改dns的时候才发现ns没有生效，原来是万恶的阿里云给暂停了。

在CentOS上搭建PHP服务器环境

安装apache:

yum install httpd httpd-devel

启动apache:

/etc/init.d/httpd start
