Microsoft已经向Win7/Win8正版用户推送Windows 10预定

今天早晨全球的Windows 7/8正版用户都收到了来自微软的Windows 10免费升级提示，用户可以在弹窗中“预订免费升级版”，与预览版本不同，用户在预订这一版本后，仍然需要等待通知，等Windows 10正式发布就可以下载3GB大小的安装包，安装后就可以平滑升级到Windows 10，目前OEM版暂时未收到推送提醒，原因未知。
