浅谈linux中的reboot和shutdown -r now区别

在linux命令中reboot是重新启动，shutdown -r now是立即停止后重新启动；虽然都为重启其实是有一定的区别的。

  1. shutdown命令可以安全的关闭或重启linux系统，它在系统关闭之前给系统上的所有登陆用户提示一条警告信息。
  2. 该命令还允许用户指定一个时间参数，可以是一个精确的时间，也可以是一个时间段。
  3. 精确时间的格式是hh:mm，表示小时和分钟，时间段由 + 和分钟数表示，系统执行该命令后会自动进行数据同步的工作。
  4. 该命令的一般格式: shutdown [选项] [时间] [警告信息]

命令中各选项的含义为:

  • -k 并不真正关机而只是发出警告信息给所有用户

  • -r 关机后立即重新启动

  • -h 关机后不重新启动

  • -f 快速关机重启动时跳过fsck

  • -n 快速关机不经过init 程序

  • -c 取消一个已经运行的shutdown

需要特别说明的是该命令只能由超级用户使用。

如:系统在5分钟后执行重新动作:#shutdown -r +10

如:系统在执行关机动作:#shutdown -h now 现在立即关闭

另外，halt命令其实就是调用shutdown -h命令，halt执行时,杀死应用进程,执行sysnc系统调用

  • -n 在关机时不会将当前的资料写入到硬盘,如果正在编辑文本没有保存那么将丢失

  • -w 不会真关机,只是将记录写入到/var/log/wtmp文档中

  • -f  强迫关机

  • -i 在关机前先把网络相关停止
