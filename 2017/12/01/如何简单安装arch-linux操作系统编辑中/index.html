如何简单安装Arch Linux操作系统？（编辑中）

Arch Linux操作系统是个有魅力的Linux发行版，系统简洁优雅，所有的一切都需要自己配置，适合对操作系统有严格要求的人，另外系统采用滚动式更新，优缺点就不说了，这里简单说一下如何简单快速安装Arch Linux操作系统。

系统要求：

  • 内存不小于512M
  • base 组要求800M存储空间
  • 不提供DVD，需要网络连接下载软件包

建立虚拟机阶段就不介绍了，这里只写出加载后的安装方法：

  1. 网络测试，输入ping www.tiankong.info，ping的通说明网络没问题；
  2. 接下来测试启动方式，输入efivar -l如果出现很多行内容说明是UEFI模式启动；
  3. 校正网络时间，输入 timedatectl set-ntp true自动联网校正；
  4. 检查磁盘名称，输入lsblk进行查看，找到你的硬盘；
  5. 进行磁盘分区，UEFI模式推荐使用cfdisk进行分区；
  6. 输入cfdisk后，选择磁盘分区为gpt；
  7. 然后推荐创建boot引导分区、/根分区、/home用户分区、swap交换分区；
  8. 本次演示分区是2G引导，10G根分区，20G主分区，8G交换分区；
  9. 首先处理引导分区，输入mkfs.vfat -F32 /dev/sda1，然后会显示版本信息；
  10. 依次格式化mkfs.ext4 /dev/sda2和mkfs.ext4 /dev/sda3分区；
  11. 最后输入mkswap /dev/sda4，然后开启swapon /dev/sda4；
  12. 格式化工作已经完成，开始进行挂在；
  13. 挂在主分区：mount /dev/sda2 /mnt；
  14. 输入mkdir /mnt/boot和mkdir /mnt/boot创建挂在文件夹；
  15. 然后挂在mount /dev/sda1 /mnt/boot和mount /dev/sda3 /mnt/home；
  16. 输入lsblk查看挂在是否正常，在MOUNTPOINT下应该4个都有名称；
  17. 编辑更新源，输入nano /etc/pacman.d/mirrorlist，保留China源；
  18. 输入pacman -Syy更新源地址；
  19. 输入pacstrap -i /mnt base base-devel net-tools进行系统安装；
  20. 含义：-i安装前确认，base-devel使用AUR，netstat和ifconfig属于net-tools工具；
  21. 输入genfstab -U /mnt >> /mnt/etc/fstab生成fstab；

以上是安装步骤，接下来进行简单配置：

  1. 输入arch-chroot /mnt /bin/bash命令Change root 到新安装的系统
  2. 输入nano /etc/locale.gen设置Locale；
  3. 保留en_US.UTF-8 UTF-8和zh_CN.UTF-8 UTF-8，并去掉前面的#注释；
  4. 输入locale-gen生成locale.gen文件；
  5. 然后使用echo LANG=zh_CN.UTF-8 >> /etc/locale.conf生成配置文件
  6. 设置上海时区ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime；
  7. 设置标准UTC时间hwclock –systohc –utc；
  8. 设置主机名echo Arch > /etc/hostname
  9. 编辑hosts文件nano /etc/hosts；
  10. 输入passwd设置root密码；
  11. 输入bootctl install安装bootloader；
  12. 拷贝示例文件cp /usr/share/systemd/bootctl/arch.conf /boot/loader/entries/；
  13. 修改/boot/loader/loader.conf；

然后

exit

umount /mnt/{boot,home}

umount /mnt

reboot
