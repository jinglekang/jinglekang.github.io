专业显卡与游戏卡我该选择哪个？

“我需要专业显卡吗？专业显卡对我的工作有很大帮助吗？”

相信很多处于专业领域的人都会遇到这个问题，现在来给大家分析一下专业显卡与游戏卡的详细介绍，相信大家看过之后会对自己的需求有些理解。

首先看专业卡和游戏卡在规格上有什么不同 这里以Quadro P5000和GTX 1080作为对比：

  1. 核心：两者都是Pascal架构的GP104核心 都拥有256bit的位宽和2560个流处理器 也就是说 两者的核心性能是几乎一模一样的 本质并没有什么区别
  2. 显存：两者都是GDDR5X显存 区别是P5000是16G显存 GTX 1080是8G 并且P5000是ECC显存
  3. 接口：P5000是4个DP+1个DVI接口 GTX1080是3个DP接口+1个HDMI接口+1个DVI接口
  4. PCB：我没拆过 不过如果两者都是公版应该是一样的 P5000就算为了稳定用料更好也不会是质的差别
  5. 驱动：这是两者最大的区别 专业卡拥有自己的专业驱动与游戏卡区分 并且两者不能互刷
  6. 外观散热方面：1080有各家自己的产品，P5000就一种没得选
  7. 价格：顶级非公GTX1080大概在4200元左右 P5000 15600元

现在来看专业卡哪方面有优势

  1. 专业驱动，这是绝对的优势，针对各种生产力软件有专门的驱动优化
  2. 大显存+ECC，对大量图形的处理更加游刃有余，ECC对个人用户和工作室几乎没有任何用处
  3. 两者双精度都有阉割，一个水平，别争了
  4. 价格，两者的价差三倍 P5000完胜(逃)

下面是应用层面：

  1. 2D程序/合成软件：代表：Adobe全家桶，对于应用CUDA的程序，比如Adobe全家桶，两者几乎没有差别 AE/PS/PR/Vegas/AI/Nuke/Axure这些都不吃显卡，并且大部分都支持DX，也就是如果你从事影视后期或者平面设计 请选游戏卡 专业卡的性价比极低 并且花钱不讨好
  2. 3D程序：代表：C4D/3DS MAX/Maya/Rhino等，这个时候专业卡就有一定的优势了，因为它是用来设计或者说创造图形的，而游戏卡是用来显示图形的，并且很多软件都有专业驱动的专门优化，胜负自然明确，Maya应该是这里软件最吃显卡的 较新版本的C4D对游戏卡有一定改进 并且现在几乎所有专业软件对游戏卡都在不断推进优化 专业卡和游戏卡的差距越来越小
  3. Deep Learning：俩没区别，但是价格摆在那里

PS：说下自己常用的软件 PS/AI/AE/C4D/Maya（我估计已经可以猜出来我干啥的了…），使用GTX 1080在工作中没有感觉有什么明显卡顿

总结：

  1. 如果你从事平面设计或者影视后期，并且预算不够的话，请买游戏卡
  2. 如果你是重度3D用户(注意“重度” )并且预算很足，可以选专业卡
  3. 如果预算不够并且不是从事电影特效这种需要超高3D需求的，就选游戏卡

在绝大多数工作中 一张GTX 1060及以上显卡就已经足够 毕竟一张P5000都可以买两个Titan XP了，P6000就更美好了

还有，专业卡不推荐农企的！！！包括最新的Vega FE，专业卡最要命的驱动做不好，还是等等吧。
