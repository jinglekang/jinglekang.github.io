某客户的组织架构同步方案设计

森松组织架构同步方案

自动同步内容

  • 岗位新增
  • 新员工入职
  • 员工部门调动
  • 员工岗位变动
  • 员工工号变动
  • 员工档案

备注： 1. 新员工入职：入职部门不存在则需要手动添加部门 2. 员工部门调动：调动部门不存在则需要手动添加部门

手动处理内容

  • 新增单位
  • 新增部门
  • 员工跨公司调动
  • 员工离职

业务流程

  1. 点击 [ 手动同步组织架构 ] 按钮
  2. 清空 [ MDM_COMP ] ， [ MDM_DEPT ] ， [ MDM_POST ]
  3. 从MDM数据库获取 [ 单位列表 ] 存入OA数据库 [ MDM_COMP ]
  4. 从MDM数据库获取 [ 部门列表 ] 存入OA数据库 [ MDM_DEPT ]
  5. 从MDM数据库获取 [ 岗位列表 ] 存入OA数据库 [ MDM_POST ]
  6. 根据OA的 [ 单位编码 ] 遍历MDM的 [ 部门档案 ]
  7. 8.

数据库表设计

  • MDM_COMP 公司表
  • COMPCODE 公司编码
  • COMPNAME 公司名称
  • MDM_DEPT 部门表
  • DEPTCODE 部门编码
  • DEPTNAME 部门名称
  • COMPCODE 公司编码
  • MDM_POST 岗位表
  • JOBCODE 岗位编码
  • JOBNAME 岗位名称
  • JOBSERIESCODE 岗位序列编码
  • COMPCODE 公司编码
  • MDM_TODO 待办表
  • CODE 代码
  • NAME 名称
  • TYPE 类型
  • OLDCODE 原代码
  • OLDNAME 原名称
  • NEWCODE 新代码
  • NEWNAME 新名称
